## Deployment

This project is configured for Vercel deployment. Simply connect your GitHub repository to Vercel and it will automatically detect the Vue/Vite configuration.

### Build Settings (Auto-detected)

- **Build Command**: `npm run build`

- **Install Command**: `npm install`
